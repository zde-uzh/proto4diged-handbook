# Inhaltliche Annotation

[mir ist begrifflich nicht ganz klar, wie sich inhaltliche Annotation und Kommentar genau unterscheiden bzw zueinander verhalten - hier sollten wir Missverständnisse zuerst durch eine Definition ausräumen]: #

Unter der _inhaltlichen Annotation_ verstehen wir die Auszeichnung eines Wortes oder einer Wortfolge hinsichtlich ihrer semantischen Referenzen. Diese können sich auf denselben Text (intrareferenziell), andere Texte/Werke (intertextuell) oder beliebiege Fakten (kontextuell) beziehen, das heisst z.B. historische, geografische, biographische Kontexte.
Im Gegensatz zur Kommentierung (siehe "commenting.md") ist eine inhaltliche Annotation ein einfacher Verweis, der selbsterklärend sein sollte oder zumindest erklärt wird durch die Weiterleitung zu einer editionsinternen Ressource (z.B. einer Karte oder einem Personeneintrag) oder einer externen Ressource (z.B. einer Normdatenbank wie GND oder Metagrid)

Die inhaltliche Annotation kann idealerweise in der gleichen Plattform wie die Kommentierung gemacht werden.

Für diverse Konnektoren, z.B. GND, bietet TEI-Publisher ein Modul, das gleichzeitig mit der Annotation einen Index ansammeln kann.
