# TEI-Modell

Etablierter Standard die Auszeichnung digitaler Editionen ist TEI/XML (Text Encoding Initiative).
Danach herrscht aber Uneinigkeit, wie die einzelnen Tags verwendet werden sollen.

Der beste Ausgangspunkt ist, wenn ein paralleles Projekt besteht, dessen Schema auf das neue Projekt übertragen werden kann.
Weitere Orientierungspunkte sind das [DTA-Basisformat](https://www.deutschestextarchiv.de/doku/basisformat/index.html) oder [TEI simple](https://teic.github.io/TEI-Simple/).
