# Using Transkribus

## Editing Lines

### Joining Lines

**Desktop Client**:

To select fragments of a line, click with `Cmd` (Mac) or `Ctrl` (other systems) to make multiple selections. It does not matter whether you select the line regions (light blue) or the baselines (dark blue).

Once the parts are selected, they can be merged using the "Merge the selected spaces" button (merged arrow).

Any text parts contained within will be merged into the new line. Annotations are lost in the process.

### Deleting Lines

**Desktop Client**:

Lines can be deleted simply by selecting them and pressing the 'Delete' button.

### Adding Lines

**Desktop Client**:

Baselines can be added using the `+ BL` key:

- Activate the mode via the button
- Click at the bottom left edge of the line
- Set individual points in between by clicking
- End the line by double-clicking

Transkribus will automatically add the required line region.

### Extending Lines

**Desktop Client**

Select the light blue area of the line. The lines can be adjusted either via the corners or by moving with the cursor.  
Then, the baselines should be adjusted by moving individual points.

## Editing Text Regions

Text regions serve as the foundation for paragraphs, titles, footnotes, and other text elements of the same level.

### Separating Text Regions

The scissors icon can be used to access the functionality for separating regions.  
Especially if the lines are slightly slanted, separating them with a line is most suitable.

### Connecting Text Regions

Similar to joining lines.

## File Upload

The best approach for working with Transkribus is to organize the facsimiles by document in folders.  
Documents can be uploaded individually via the client or the web version.  
For uploading multiple documents, using an FTP client like FileZilla is recommended.
