# Usage of Transkribus

## Editing of lines

### Joining lines

**Desktop Client**:

Select the fragments of a line by clicking with `Cmd` (Mac) or `Ctrl` (other systems) for multiple selection.
It makes no difference whether you select the regions of the lines (light blue) or the base lines (dark blue).

Once the parts have been selected, they can be merged using the "Merge the selected spaces" button (merged arrow).

Any text parts contained are merged in the new line. Annotations are lost in the process.

### Deleting lines

**Desktop Client**

Lines can be deleted simply by selecting them and pressing the 'Delete' button.

### Adding lines

**Desktop Client**

Baselines can be added using the `+ BL` key:

- Activate the mode via the button
- Click at the bottom left edge of the line
- Set individual points in between by clicking
- End the line by double-clicking

Transkribus adds the required line region.

### Extending lines

**Desktop Client**

Selection of the light blue area of the line. The lines can be changed either via the corners or via the cursor for moving.
The base lines should then be adjusted. This is done by moving individual points.

## Editing text regions

The text regions are the basis for paragraphs, titles, footnotes and other text elements of the same level.

### Separating text regions

The scissors icon can be used to call up the functionality for separating regions.
Especially if the lines are slightly slanted, separation by a line is most suitable.

### Connecting text regions

Analogue to joining lines.

## File upload

The best way to work with Transkribus is to have the facsimiles organised by document in folders.
Documents can be uploaded individually via the client or the web version.
For uploading multiple documents, we recommend using an FTP client such as Filezille.
