# Über die Dokumentation von Workflow-Schritten

Die vorliegende Dokumentation der einzelnen Arbeitsschritte - von der Transkription bis zur Publikation - folgt der Annahme, dass sich die Logik dieser Schritte in den nächsten Jahren nicht grundsätzlich verändern wird. Es ist jedoch davon auszugehen, dass die hier dokumentierten tools und ihre features grundlegenden Veränderungen verschiedene Anpassungen erfahren werden. 

Deshalb versucht die vorliegende Dokumentation  Arbeitsabläufe so nachzuzeichnen, dass nicht die konkrete Handlung, sondern die ihr unterliegende Logik mitsamt ihren Herausforderungen im Vordergrund steht.
